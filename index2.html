Projetos de Dados e BI

Desenvolvimento de Modelo de Priorização de Cargas

Utilizei Python e conceitos de Estatística para realizar uma análise combinatorial de Containeres com diversos SKUs para decidir a prioridade de Recebimento baseado nos níveis de ruptura de estoque, ocupação do armazém e capacidade de recebimento. A ferramenta, aplicada em conjunto com outras estratégias, reduziu a ocupação média do armazém em 10%, enquanto o nível de ruptura para cada SKU foi reduzido em até 5 dias.

Dashboards de Vendas para empresa de Ecommerce

Utilizei Power BI e a linguagem DAX para criar dois dashboards de vendas para uma empresa de Ecommerce. Levei em consideração a relevância das informações para tomada de decisão, assim como a versatilidade para mostrar diversas visões e filtros. Dois dashboards foram desenvolvidos, um deles uma réplica do que foi dado como uma imagem, e o outro foi construído do zero, utilizando medidas criadas com DAX e ferramentas de front-end.
